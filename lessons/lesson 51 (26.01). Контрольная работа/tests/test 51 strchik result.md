# Вопрос 1. Тема 1: Условные конструкции и циклы в Python
  
Какие значения в Python считаются ложными (falsy)?
  
Варианты ответов:
1) ❌ Только значение 0
2) Только None
3) Только отрицательные числа
4) Значения, приводимые к False
5) Только пустые строки
  
### ИИ анализ
Выбран неверный вариант. В Python к ложным (falsy) относятся не только 0, но и пустые коллекции (список, строка, словарь), `None`, и `False`. Правильный ответ — "Значения, приводимые к False", так как это общее определение для всех ложных значений.  Важно запомнить полный список falsy-значений для корректной работы с условиями.  

# Вопрос 2. Тема 1: Условные конструкции и циклы в Python
  
Зачем используется `elif`?
  
Варианты ответов:
1) ❌ Для обработки исключений
2) Для вывода текста
3) Для создания функции
4) Для остановки цикла
5) Для добавления нового условия
  
### ИИ анализ
Выбран неверный вариант. Ключевое слово `elif` используется для проверки дополнительных условий после `if`, если предыдущие условия оказались ложными. Обработка исключений выполняется с помощью `try/except`.  Для написания сложных условных ветвлений важно правильно использовать цепочку `if-elif-else`.

1) Когда известен набор значений
2) ❌ Когда нужно повторять бесконечно

# Вопрос 4. Тема 1: Условные конструкции и циклы в Python
  
В каком случае удобнее использовать `while`?
  
Варианты ответов:
1) Когда есть список значений
2) ✅ Когда условие важно каждый раз
3) Когда есть range
4) Когда нужен словарь
5) Когда нужна сортировка
  
### ИИ анализ
Выбран верный вариант. Цикл `while` используется, когда повторение зависит от условия, которое проверяется перед каждой итерацией. Это отличается от `for`, который перебирает заранее заданные элементы.  Этот выбор фундаментален для реализации циклов с неизвестным заранее количеством шагов.  

# Вопрос 5. Тема 1: Условные конструкции и циклы в Python
  
Что делает `break`?
  
Варианты ответов:
1) Перезапускает цикл
2) ✅ Полностью прерывает цикл
3) Пропускает итерацию
4) Переходит в else блока
5) Останавливает программу
  
### ИИ анализ
Выбран верный вариант. Оператор `break` немедленно завершает выполнение самого внутреннего цикла, в котором он находится. Для пропуска итерации используется `continue`.  Правильное применение `break` позволяет эффективно выходить из циклов при выполнении определенного условия.  

# Вопрос 6. Тема 1: Условные конструкции и циклы в Python
  
Что делает `continue`?
  
Варианты ответов:
1) Возвращает значение
2) ✅ Пропускает текущую итерацию
3) Завершает программу
4) Завершает цикл полностью
5) Переходит к else блока
  
### ИИ анализ
Выбран верный вариант. Оператор `continue` прерывает текущую итерацию цикла и переходит к проверке условия для следующей итерации. Это полезно для пропуска определенных шагов без полного выхода из цикла.  Путать `continue` с `break` — распространенная ошибка, которую важно избегать.
