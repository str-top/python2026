# Описание задачи

Необходимо написать программу, которая будет анализировать данные о продажах продуктов в различных магазинах, используя данные в формате CSV и JSON. Программа должна выполнить несколько операций:

- прочитать CSV-файл с данными о продажах продуктов в различных магазинах. Каждый продукт имеет цену, количество проданных единиц и дату продажи;
- принять на вход дату и сгенерировать отчёт о количестве проданных продуктов в указанный день;
- вывести на экран информацию о магазине с наибольшими продажами за день (по выручке);
- сохранить итоговый отчёт о продажах продуктов в новый JSON-файл.

## Указания по использованию технологий и методов

Для анализа данных используйте стандартные операторы Python (циклы, условные операторы, списковые выражения и т.д.).

## Пример входных данных

### `sales_data.csv`:

```csv
Date,Store,Product,Price,Quantity
2024-10-01,Store_A,Apple,2.5,100
2024-10-01,Store_A,Banana,1.8,200
2024-10-01,Store_B,Apple,2.5,150
2024-10-01,Store_B,Orange,2.0,180
2024-10-02,Store_A,Apple,2.5,120
2024-10-02,Store_A,Banana,1.8,250
2024-10-02,Store_B,Apple,2.5,130
2024-10-02,Store_B,Orange,2.0,170
```

### Входные данные от пользователя:

```
Введите дату для отчетности (формат YYYY-MM-DD): 2024-10-01
```
